Table QR setup

Guests scan a QR on their table — no manual table entry. Orders on WhatsApp show Table #5 for admin.

How to create QRs

  1. Use any QR generator (qr-code-generator.com, Canva, etc.)
  2. One QR per table — use the link format below
  3. Print and place on each table
index.html?table=1
Table 1 → ?table=1 → admin sees Table #1 Table 12 → ?table=12 → admin sees Table #12 Patio → ?table=Patio → admin sees Table #Patio
